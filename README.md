# TerraNugget SMP

The TerraNugget SMP Pack was originally created for a small SMP-Project with friends to achieve a Vanilla+ experience but with the world generation of Terralith + Tectonic. There are also new structures, performance, graphic and QoL mods to improve the gameplay. Besides that it adds multiplayer features like Simple Voice Chat and more to make SMP more fun.

**No blocks or items are added to the game!**

## Installation
This modpack can be found on [**Modrinth**](https://modrinth.com/project/terranugget-smp). For the best experience, use the [Modrinth App](https://modrinth.com/app) to install it and switch versions.

**Usage on Servers** 

It is recommended to use [mrpack-install](https://github.com/nothub/mrpack-install) to create a server for this modpack.
