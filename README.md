# TerraNugget SMP

The TerraNugget SMP Pack was originally created for a small SMP-Project with friends to achieve a Vanilla+ experience. It focuses on enhancing world generation, adding new structures, and integrating performance, graphical, and Quality of Life (QoL) mods to improve gameplay. It also includes multiplayer features to make SMP more enjoyable.

**No modded blocks or items are added to the game!**

## Features

Based on the included mods, players can expect the following additions and tweaks:

### World Generation & Structures
* **Overworld:** Modified terrain and biome generation using **Terralith** and **Tectonic**.
* **Dimensions:** Updated Nether and End generation via **Amplified Nether** and **Nullscape**.
* **Structures:** Overhauled vanilla structures (such as Desert Temples, Ocean Monuments, Pillager Outposts, and Mansions) using the **Dungeons and Taverns** mods.

### Multiplayer 
* **Proximity Chat:** Integrated **Simple Voice Chat** for in-game communication.
* **Host Anywhere:** Includes **e4mc** to easily share single-player worlds with friends.
* **Server Utilities:** Features **Ledger** for block logging and **Universal Graves** to handle player deaths.

### Performance
* **Core Optimization:** Includes standard Fabric performance mods such as **Sodium**, **Lithium**, and **FerriteCore**.
* **World Generation:** Fast chunk calculation using **C2ME** and **NoisiumForked**.

### Visuals & Audio
* **Graphics:** Shader support via **Iris Shaders** and connected textures through **Continuity**.
* **Models & Animations:** Updated entity models and player features via **Fresh Animations** and **3D Skin Layers**.
* **Audio:** Adjusted acoustics and reverberation using **Sound Physics Remastered**.

### Quality of Life & Technical Tools
* **Mapping:** Navigation support with **Xaero's Minimap** and **Xaero's World Map**.
* **Technical Mods:** Client-side tools for technical players, including **Litematica**, **MiniHUD**, **Tweakeroo**, and **Carpet**.
* **Building:** **Axiom** is included for advanced creative building.

## Installation
This modpack can be found on [**Modrinth**](https://modrinth.com/project/terranugget-smp). For the best experience, use the [Modrinth App](https://modrinth.com/app) to install it and switch versions.

**Usage on Servers** It is recommended to use [mrpack-install](https://github.com/nothub/mrpack-install) to create a server for this modpack.
